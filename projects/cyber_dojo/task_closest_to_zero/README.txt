Closest To Zero
---------------
Given a list of integers find the closest to zero.
If there is a tie, choose the positive value.




------------------------
Additional statements for implementations using Python:

The given list may actually be any iterable, such as a list or a tuple, or a generator.

Some examples:
Input                      Output
[-3]                          -3
[15, 8]                        8
[15, 7, 22]                    7
[5, 7, -3, -8]                -3
[5, 7, -5, -8]                 5
(5, 7, -5, -8)                 5
(x for x in range(4, 14, 3))   4
