First Longest Word
-------------------

Write a program that returns the first longest word in a string.

Requirements:
* Ignore punctuation.
* Words can contain letters and numbers.


Test examples:
   ""                        -> ""
   "&!="                     -> ""
   "Have a nice time!"       -> "Have"
   "Have a&!! nice weekend"  -> "weekend"
   "Are you self-aware?"     -> "self-aware"
   "I really like AnomGirl91 -> "AnomGirl91"
